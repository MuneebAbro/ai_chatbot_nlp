üé® Color Customization

Add primary and secondary colors for perfect contrast:

<script src="https://bitpack-widget.vercel.app/widget/embed/test?color=%23FFC0CB&secondary=%23000000"></script>

Available color parameters:

  • color - Primary color for buttons and backgrounds
  • secondary - Text and icon color (for contrast)
  • theme - Alternative to color
  • text - Alternative to secondary
  • icon - Alternative to secondary
  • position - Widget position (bottom-right, bottom-left, top-right, top-left)
